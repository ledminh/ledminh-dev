# Minh Le Portfolio

Personal website that introduces me, shares a timeline and contact details,
hosts a small blog, and documents my learning journal.

## What it does

- Presents a landing page and About page
- Lists a timeline and contact info
- Publishes a simple blog page
- Shows learning journal entries written in MDX

## Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Content locations

- Landing page: `app/page.tsx`
- About page content: `app/about/description.mdx`
- About timeline: `app/about/timeline.json`
- About contact info: `app/about/contact.json`
- Blog page: `app/blog/page.mdx`
- Learning journal intro: `app/learning-journal/description.mdx`
- Learning journal entries: `app/learning-journal/entries/*.mdx`
